# Nextcloud Sync — Project Rules
## Operative Task-Arbeit

- **Repo-Wahrheit vor Chat-Wahrheit.** Tasks, Reports, Übergaben gehören ins Repo.
- Bei operativer Task-Arbeit in `docs/tasks/T-xxxx_<kurzname>/` arbeiten.
- Standard-Task-Struktur: `00_brief.md`, `01_scope.md`, `02_done_when.md`, `03_constraints.md`, `04_impl_report.md`, `05_validation_report.md`
- `docs/tasks` = operative Übergabeschicht
- `docs/reviews` = kanonische repo-spezifische Reviews
- Berichte: kurz, konkret, überprüfbar
- Team-Regeln gelten für ChatGPT, Claude und Cursor gleichermaßen
